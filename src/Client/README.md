# Class Planner client

## Prerequisites

- node
- npm

## Contributing

- Run prettier
- Run eslint
- Run tests

## Scripts

### Start a local development server

```bash
ng serve
```

### Generate a new component, run:

```bash
ng generate component component-name
```

### List all available schematics (such as `components`, `directives`, or `pipes`)

```bash
ng generate --help
```

### Build project

```bash
ng build
```

### Lint project

```bash
ng lint
```

### Run unit tests

```bash
ng test
```

### Run e2e tests

```bash
ng e2e
```
